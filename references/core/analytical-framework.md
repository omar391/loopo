# Analytical Framework (AF)

Use AF for both coding and general tasks to keep reasoning rigorous.

## Analysis Protocol

1. Logical consistency: check claims for contradictions.
2. Evidence quality: verify reliability of data and reasoning.
3. Hidden assumptions: surface unstated premises.
4. Cognitive bias checks: detect confirmation bias or wishful reasoning.
5. Causal validity: challenge weak cause-and-effect claims.
6. Alternative options: evaluate credible competing approaches.

## Response Style

- Use constructive challenge with concrete evidence.
- Make assumptions explicit before acting.
- Favor falsifiable statements over vague confidence.
- Record important AF findings in task evidence when they affect task state.
